' Core relationships
World --> System : manages
System --> Entity : processes
Entity --> Component : contains
Component --> System : registers with

note top of World
  **World Layer**
  - Manages game loop
  - Coordinates all systems
  - Update/Render cycle
end note

note right of System
  **System Layer**
  - Processes entities
  - Domain-specific logic
  - Update per frame
end note

note bottom of Entity
  **Entity Layer**
  - Container for components
  - Manages lifecycle
  - Active/Inactive state
end note

note left of Component
  **Component Layer**
  - Extends entity functionality
  - Registers with systems
  - Lifecycle management
end note
